# BOOTC installer

### This is a very simple containerfile based on Fedora Minimal, designed for easy rebasing to an image using bootc.

Here, you can rebase to any Desktop or Bootc Image. the primary purpose of this image is so then a "Universal" installer in the form of an ISO can be created.
